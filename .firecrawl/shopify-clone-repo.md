## Configuration and Setup

[Permalink: Configuration and Setup](https://github.com/sh3yash/Shopify_Clone#configuration-and-setup)

In order to run this project locally, simply fork and clone the repository or download as zip and unzip on your machine.

- Open the project in your prefered code editor.
- Go to terminal -> New terminal (If you are using VS code)

In the first terminal

```
$ cd react-shopify-landing
$ npm install
$ npm run dev
```

- [React Vite](https://vitejs.dev/guide/) \- Improved website speed.
- [React Hooks](https://reactjs.org/docs/hooks-intro.html) \- For managing and centralizing application state.

## About

A Shopify clone built on React offers a dynamic, scalable e-commerce platform with a responsive design, customizable themes, and advanced product management. Key features include streamlined checkout, a robust admin dashboard, SEO tools, and user authentication. React ensures high performance and extensibility for a modern online store.

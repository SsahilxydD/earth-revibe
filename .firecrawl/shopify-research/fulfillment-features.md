# Overview of order fulfillment and shipping features on Shopify

With Shopify, you have access to a comprehensive suite of built-in order fulfillment and management features that allow you to efficiently process, fulfill, and ship customer orders. From initial setup to final delivery, these features can help you create a seamless fulfillment workflow without requiring third-party apps.

## Shipping setup

Before you can fulfill orders, you need to configure your shipping and fulfillment settings. Set up locations, shipping profiles, zones, and rates. Configure delivery methods, notifications, and international shipping support to create a complete fulfillment foundation.

Review the following tables to learn more about shipping setup features.

### Locations and shipping rates

| Feature | Description |
| --- | --- |
| [Locations](https://help.shopify.com/en/manual/fulfillment/setup/locations) | Set up and manage physical locations where you store inventory and fulfill orders. You can configure multiple locations to optimize shipping costs and delivery times for your customers. |
| [Shipping profiles](https://help.shopify.com/en/manual/fulfillment/setup/shipping-profiles) | Create custom shipping profiles to group products with similar shipping requirements. Use profiles to apply different shipping rules based on product type, weight, or destination. |
| [Shipping zones](https://help.shopify.com/en/manual/fulfillment/setup/shipping-rates/shipping-zones) | Create geographic shipping zones to group countries and regions with similar shipping rates and delivery options. Control where you ship and apply zone-specific pricing and restrictions. |
| [Shipping rates](https://help.shopify.com/en/manual/fulfillment/setup/shipping-rates) | Configure shipping rates to control how much you charge for shipping. Set up flat rates, weight-based rates, or price-based rates for different regions and delivery options. |
| [Transit time](https://help.shopify.com/en/manual/fulfillment/setup/shipping-rates/transit-time) | Add transit time to your flat shipping rates to display transit time to your customers at checkout. Transit time is the time frame from when your shipping carrier collects the shipment from you, to when the shipment arrives at your customer's address. |
| [Carrier-calculated shipping rates](https://help.shopify.com/en/manual/fulfillment/setup/shipping-rates/third-party-carrier-calculated-shipping) | Connect your own account with an eligible shipping carrier to your Shopify admin to display real-time shipping rates from carriers such as UPS, FedEx, and USPS directly at checkout. Rates are calculated based on package dimensions, weight, and destination. |
| [Backup shipping rates](https://help.shopify.com/en/manual/fulfillment/setup/shipping-rates/backup-rates) | Set up backup shipping rates that are used when a shipping carrier or a shipping app can't display shipping rates at checkout. |

### Delivery methods

| Feature | Description |
| --- | --- |
| [Pickup in store](https://help.shopify.com/en/manual/fulfillment/setup/delivery-methods/pickup-in-store) | Let customers collect their orders directly from your physical locations. Display available pickup times and locations during checkout, and send ready-for-pickup notifications. |
| [Local delivery](https://help.shopify.com/en/manual/fulfillment/setup/delivery-methods/local-delivery) | Offer delivery within specific geographic areas around your store locations. Set delivery zones, fees, and time windows to provide convenient local delivery options. |
| [Pickup points](https://help.shopify.com/en/manual/fulfillment/setup/delivery-methods/pickup-points) | Let customers choose convenient pickup locations for their shipments. Integrate with carrier pickup networks to provide alternative delivery options. |

Delivery methods features and their descriptions

### Additional shipping setup features

| Feature | Description |
| --- | --- |
| [Packages](https://help.shopify.com/en/manual/fulfillment/setup/packaging/setting-up-packages) | Define custom package sizes and weights to ensure accurate shipping calculations at checkout and when buying shipping labels. Assign a package to your products and product variants for single item orders. |
| [Processing time and delivery dates](https://help.shopify.com/en/manual/fulfillment/setup/processing-time-and-delivery-dates) | Configure processing times for different products and locations to provide accurate delivery estimates. Set business days, holidays, and cutoff times for order processing. |
| [Shop Promise](https://help.shopify.com/en/manual/fulfillment/setup/shop-promise) | Display estimated delivery dates to customers during checkout and throughout their order journey. Help set accurate expectations and increase customer confidence. |
| [Fulfillable inventory](https://help.shopify.com/en/manual/fulfillment/setup/fulfillable-inventory) | Control which inventory can be sold based on fulfillment capabilities and shipping zones. Prevent overselling by limiting availability to areas that you can actually sell to. |
| [Order routing](https://help.shopify.com/en/manual/fulfillment/setup/order-routing) | Configure automatic rules to determine which location fulfills each order. Route orders based on inventory availability, shipping cost, proximity to customer, or custom business logic. |
| [Order notifications](https://help.shopify.com/en/manual/fulfillment/setup/notifications) | Customize email and SMS notifications that are automatically sent to customers and staff throughout the order fulfillment process. Personalize templates with your branding and specific messaging. |
| [Carbon neutral shipping](https://help.shopify.com/en/manual/fulfillment/setup/planet) | Make your shipments carbon neutral using the Planet app, which funds innovative companies that remove carbon dioxide (CO2) from the atmosphere and store it. |
| [International shipping support](https://help.shopify.com/en/manual/international/shipping/hs-codes) | Handle international shipments with automated customs documentation, Harmonized System (HS) codes, and duty calculations. Simplify cross-border commerce with built-in compliance tools. |

## Order management and processing

After customers place orders, you can search, process, and manage orders using built-in order management tools. Handle order details, payments, refunds, fraud protection, printing, and automation to efficiently prepare orders for fulfillment.

Review the following table to learn more about order management and processing features.

| Feature | Description |
| --- | --- |
| [Searching and filtering orders](https://help.shopify.com/en/manual/fulfillment/managing-orders/viewing-orders) | Find specific orders quickly using built-in search and filtering options. Search by order number, customer name, email, product, fulfillment status, and custom criteria. |
| [Managing order details](https://help.shopify.com/en/manual/fulfillment/managing-orders/managing-order-details) | View and manage comprehensive order information including customer details, shipping addresses, payment information, and order history. Access all order data in one centralized location. |
| [Creating draft orders and invoices](https://help.shopify.com/en/manual/fulfillment/managing-orders/create-orders) | Create orders manually for customers through phone, email, or in-person sales. Generate professional invoices, customize pricing, apply discounts, and convert drafts to completed orders when ready. |
| [Editing orders](https://help.shopify.com/en/manual/fulfillment/managing-orders/editing-orders) | Modify existing orders by adding or removing products, updating quantities, changing shipping addresses, or adjusting prices. Handle order changes efficiently without canceling and recreating orders. |
| [Validating customer shipping address](https://help.shopify.com/en/manual/fulfillment/managing-orders/validating-order-address) | Automatically confirm and standardize customer shipping addresses to reduce delivery failures and ensure accurate shipping calculations. Prevent common address errors before fulfillment. |
| [Refunding orders](https://help.shopify.com/en/manual/fulfillment/managing-orders/refunding-orders) | Process full or partial refunds for completed orders. Handle refunds for products, shipping, taxes, and fees while automatically updating inventory and sending customer notifications. |
| [Protecting orders from fraud](https://help.shopify.com/en/manual/fulfillment/managing-orders/protecting-orders) | Use built-in fraud analysis tools to identify and review potentially fraudulent orders. Protect your business with automatic risk assessments and manual review workflows. |
| [Printing orders](https://help.shopify.com/en/manual/fulfillment/managing-orders/printing-orders) | Generate and print packing slips, invoices, and other order documents to streamline warehouse operations. Customize document formats and automate printing workflows. |
| [Exporting orders](https://help.shopify.com/en/manual/fulfillment/managing-orders/exporting-orders) | Export order data for analysis, accounting, or external systems. |
| [Automating orders with Shopify Flow](https://help.shopify.com/en/manual/shopify-flow) | Create automated workflows for common order processing tasks. Automatically tag orders, route fulfillment, update inventory, and trigger actions based on order conditions. |
| [Canceling orders](https://help.shopify.com/en/manual/fulfillment/managing-orders/canceling-orders) | Cancel unfulfilled orders before processing. Automatically handle payment reversals, inventory adjustments, and customer notifications when orders need to be canceled. |

## Shipping labels and carrier services

You can purchase shipping labels, print shipping documents, manage carrier services, and handle shipping logistics directly from your Shopify admin. These integrated shipping features can help streamline your shipping operations with professional carrier services, insurance options, and international shipping capabilities.

Review the following table to learn more about shipping labels and carrier services.

| Feature | Description |
| --- | --- |
| [Buying shipping labels individually](https://help.shopify.com/en/manual/fulfillment/fulfilling-orders/shipping-labels/buying-labels/buying-labels) | Purchase and print shipping labels directly from your Shopify admin. Select carrier services, confirm package details, and print labels instantly for individual shipments. Available in select countries. |
| [Buying shipping labels in bulk](https://help.shopify.com/en/manual/fulfillment/fulfilling-orders/shipping-labels/buying-labels/buying-labels) | Purchase shipping labels for multiple orders simultaneously to save time and improve efficiency. Process bulk shipments with consistent carrier selection and packaging options. Available in select countries. |
| [Shipping carrier services](https://help.shopify.com/en/manual/fulfillment/fulfilling-orders/shipping-labels/shipping-carriers) | Access multiple shipping carriers through a single integration. Compare rates, delivery times, and services across different carriers to find the best option for each shipment. |
| [Shipping insurance](https://help.shopify.com/en/manual/fulfillment/fulfilling-orders/shipping-labels/shipping-insurance) | Protect shipments with optional shipping insurance coverage. Automatically calculate insurance costs based on order value and handle claims processing. |
| [Printing shipping labels and documents](https://help.shopify.com/en/manual/fulfillment/fulfilling-orders/shipping-labels/managing-labels/printing-shipping-documents) | Print shipping labels, customs forms, and packing slips individually or in bulk using various paper formats. Customize store names on labels and combine shipping documents for efficient printing workflows. |
| [Changing and voiding shipping labels](https://help.shopify.com/en/manual/fulfillment/fulfilling-orders/shipping-labels/managing-labels/voiding-labels) | Void unused shipping labels directly from your Shopify admin and get credited back to your account. |
| [Buying DDP (Delivered Duty Paid) shipping labels](https://help.shopify.com/en/manual/fulfillment/fulfilling-orders/shipping-labels/buying-labels/ddp-ddu) | Purchase Delivered Duty Paid shipping labels that include international duties and import tax, which provides customers with transparent, prepaid international shipping costs. |
| [Scheduling pickups, creating manifests, and tracking shipments](https://help.shopify.com/en/manual/fulfillment/fulfilling-orders/shipping-labels/managing-labels) | Schedule carrier pickups, generate shipping manifests, and track all your shipping labels in one place. Streamline your shipping operations with end-to-end shipment management. |
| [Submitting insurance claims](https://help.shopify.com/en/manual/fulfillment/fulfilling-orders/shipping-labels/shipping-carriers/shipping-claims) | File claims for lost or damaged shipments directly from your Shopify admin. |

## Order fulfillment

You can choose from multiple fulfillment methods to handle different business models and order volumes. Fulfill orders manually or in bulk. Use fulfillment holds, third-party services, or custom workflows to coordinate with fulfillment services regardless of your store's location.

Review the following table to learn more about order fulfillment features.

| Feature | Description |
| --- | --- |
| [Generating pick lists](https://help.shopify.com/en/manual/fulfillment/managing-orders/printing-orders/shopify-order-printer/pick-list) | Configure and print pick lists based on your picking strategy. |
| [Marking orders as fulfilled individually](https://help.shopify.com/en/manual/fulfillment/fulfilling-orders/single-fulfillment) | Fulfill individual orders manually by marking items as fulfilled, adding tracking information, and sending customer notifications. Ideal for small order volumes or custom fulfillment needs. |
| [Marking orders as fulfilled in bulk](https://help.shopify.com/en/manual/fulfillment/fulfilling-orders/bulk-fulfillment) | Process multiple orders simultaneously to save time and improve efficiency. Select multiple orders and fulfill them all at one time with consistent tracking and notification settings. |
| [Marking orders as on hold](https://help.shopify.com/en/manual/fulfillment/fulfilling-orders/holding-fulfillments) | Place temporary holds on order fulfillment for review, inventory verification, or fraud prevention. Manage holds systematically and release them when conditions are met. |
| [Shopify Fulfillment Network](https://help.shopify.com/en/manual/fulfillment/shopify-fulfillment-network) | Use Shopify's end-to-end fulfillment service with strategically located warehouses. Shopify Fulfillment Network (SFN) handles storage, packing, and shipping with real-time inventory tracking. |
| [Third-party fulfillment services](https://help.shopify.com/en/manual/fulfillment/fulfilling-orders/app-fulfillment) | Connect with third-party logistics providers and fulfillment services that integrate directly with Shopify. Automatically route orders to external fulfillment centers. |
| [Custom fulfillment workflows](https://help.shopify.com/en/manual/fulfillment/fulfilling-orders/custom-fulfillment) | Create customized fulfillment processes for unique business requirements. Handle dropshipping, print-on-demand, or custom manufacturing workflows. |

## Returns and post-fulfillment support

After orders are fulfilled and delivered, you can manage the post-purchase experience with comprehensive tools. Handle returns and exchanges, create return shipping labels, activate self-serve returns, and provide delivery tracking through the Shop app to maintain customer satisfaction.

Review the following table to learn more about returns and post-purchase support features.

| Feature | Description |
| --- | --- |
| [Delivery tracking with Shop](https://help.shopify.com/en/manual/online-sales-channels/shop/delivery-tracking) | Provide customers with real-time order tracking, delivery notifications, and live map tracking through the Shop app. Reduce support inquiries about order status and improve first-time delivery success rates. |
| [Self-serve returns](https://help.shopify.com/en/manual/fulfillment/managing-orders/returns/self-serve-returns) | Allow customers to initiate returns and exchanges independently through their customer accounts. Provide a branded return portal with automated return authorization and tracking. |
| [Return rules and return policy](https://help.shopify.com/en/manual/fulfillment/managing-orders/returns/return-rules) | Create rules for when customers can request a return and how return fees are applied to your store. |
| [Managing returns and exchanges](https://help.shopify.com/en/manual/fulfillment/managing-orders/returns) | Process customer returns and exchanges directly in the Shopify admin. Create return merchandise authorizations (RMAs), handle restocking, and manage return workflows. |
| [Buying return shipping labels](https://help.shopify.com/en/manual/fulfillment/fulfilling-orders/shipping-labels/buying-labels/return-labels) | Purchase return shipping labels directly through Shopify to simplify the returns process for customers. Provide prepaid return labels and streamline return logistics. |
